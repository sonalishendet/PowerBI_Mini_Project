# PowerBI_Mini_Project on procurement and spend dataset #

End to End Data Analytics Project (Power BI)

This repository contains materials including Power BI Dashboard Creation

**This mini project represents a procurement and spend dataset suitable for:**

1.Spend analysis & cost optimization

2.Vendor performance evaluation

3.Category-wise and BU-wise procurement insights

4.Power BI / dashboard reporting
